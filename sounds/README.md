# Sound Files for Liar's Gun

This directory should contain the following sound files:

## Required Sound Files:
1. **bar-ambience.mp3** - Background bar atmosphere (looping)
2. **card-shuffle.mp3** - Card dealing/shuffling sound
3. **card-flip.mp3** - Card reveal sound
4. **bluff-call.mp3** - Dramatic bluff call sound
5. **gun-shot.mp3** - Gun firing sound
6. **gun-click.mp3** - Empty chamber click sound
7. **button-click.mp3** - UI button interaction sound
8. **player-join.mp3** - Player joined notification sound
9. **game-start.mp3** - Game beginning sound

## Where to Get Sounds:
- **Free Resources**: 
  - [Freesound.org](https://freesound.org/)
  - [Zapsplat.com](https://www.zapsplat.com/)
  - [Mixkit.co](https://mixkit.co/free-sound-effects/)
  
- **Search Terms**:
  - Bar ambience: "bar atmosphere", "pub ambience", "restaurant background"
  - Cards: "card shuffle", "poker cards", "card flip"
  - Gun: "revolver shot", "gun click", "pistol"
  - UI: "button click", "interface beep", "notification"

## Placeholder Files:
Currently, the game will work without sound files (audio elements will fail silently).
You can add proper sound files later by placing them in this directory.

## File Format:
All files should be in MP3 format for best browser compatibility.
